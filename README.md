# Testimonial App

A React + Vite application where users can view testimonials and submit their own.

## Features

* View existing testimonials
* Submit new testimonials
* Responsive UI
* Connected to PHP + MySQL backend (XAMPP)

## Tech Stack

* React (Vite)
* CSS
* PHP
* MySQL

## Run Locally

1. Install dependencies:
   npm install

2. Start dev server:
   npm run dev

3. Backend runs on XAMPP:
   http://localhost/api
